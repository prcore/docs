PUT /event-log/{event_log_id}
json
original-name-of-the-column required dict(str, str)

With the following format:

{
    "name": "new-name-of-the-column",
    "type": "new-type-of-the-column"
}

The type property can be chosen from the following list:

  1. Special types:
    • CASE_ID
  2. Basic types:
    • TEXT
    • NUMBER
    • BOOLEAN
    • TIMESTAMP
  3. Common types:
    • ACTIVITY
    • RESOURCE
  4. Optional types:
    • DURATION
    • COST
    • START_TIMESTAMP
    • END_TIMESTAMP

Based on previous exmaple, here we will use the following columns configuration.

{
    "Case_ID": {
        "name": "Case ID",
        "type": "CASE_ID"
    },
    "Time": {
        "name": "Date",
        "type": "TIMESTAMP"
    },
    "Action": {
        "name": "Activity",
        "type": "ACTIVITY"
    },
    "Personnel": {
        "name": "Employee",
        "type": "RESOURCE"
    }
}
